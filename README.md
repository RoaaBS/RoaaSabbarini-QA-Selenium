 # Selenium Automation Project - AutomationExercise
## 📌 Project Overview

This project is an automated test suite built using Selenium WebDriver with Java.
It follows the Page Object Model (POM) design pattern to ensure clean, maintainable, and scalable test structure.

The automation is performed on the website:
https://automationexercise.com/

## 🎯 Test Scenario Covered

The automation script covers the following end-to-end user journey:

Navigate to Home Page
Select a random product from the Products page
Open Product Details page
Update product quantity to 2
Add product to cart
View cart from modal popup
Proceed to Checkout
Place Order
Fill payment details (fake data used):
Card Number: 123456789
CVV: 100
Name: Any valid name
Expiry Date: Any valid date
Verify order confirmation message:

“Congratulations! Your order has been confirmed!”

Close the browser
## ⚙️ Technologies Used
Java
Selenium WebDriver
TestNG
Maven
Page Object Model (POM)
ExtentReports (for reporting)
